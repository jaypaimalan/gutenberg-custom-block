# Custom Card Block

A custom WordPress Gutenberg block that renders a card with:
- **Image** (media library upload)
- **Title** (rich text)
- **Description** (rich text)
- **Button** (customisable text + URL)

---

## File Structure

```
custom-card-block/
├── custom-card-block.php   ← Plugin entry point (PHP)
├── package.json            ← Node dependencies & build scripts
├── src/
│   ├── block.json          ← Block metadata (attributes, supports, etc.)
│   ├── index.js            ← Block registration
│   ├── edit.js             ← Editor UI component (JSX)
│   ├── save.js             ← Saved / front-end HTML component (JSX)
│   ├── editor.scss         ← Editor-only styles
│   └── style.scss          ← Shared styles (editor + front-end)
└── build/                  ← Auto-generated by `npm run build` (do not edit)
```

---

## Setup Instructions

### 1. Copy the plugin folder

Place the entire `custom-card-block` folder inside your WordPress installation:

```
wp-content/plugins/custom-card-block/
```

### 2. Install Node dependencies

```bash
cd custom-card-block
npm install
```

> Requires **Node.js 20+** and **npm**.

### 3. Build the block

**Development (watch mode — rebuilds on file save):**
```bash
npm start
```

**Production build:**
```bash
npm run build
```

This compiles everything from `src/` into `build/`.

### 4. Activate the plugin

1. Go to **WordPress Admin → Plugins**.
2. Find **Custom Card Block** and click **Activate**.

### 5. Use the block

1. Open any post or page in the Gutenberg editor.
2. Click **+** to open the block inserter.
3. Search for **"Custom Card Block"** (or find it under the **Design** category).
4. Insert and configure your card!

## Editor Features

| Feature | How to use |
|---|---|
| **Image** | Click "Upload Image" → opens Media Library |
| **Title** | Click and type directly on the card |
| **Description** | Click and type directly on the card |
| **Button text** | Click the button area and type |
| **Button URL** | Set in the right-hand **Inspector → Button Settings** |
| **Link target** | Choose Same Tab / New Tab in the Inspector |
| **Alt text** | Set in **Inspector → Image Settings** |

---

## Customising Styles

Edit `src/style.scss` to adjust front-end appearance.  
Edit `src/editor.scss` to adjust editor-only appearance.  
Run `npm run build` after any change.
